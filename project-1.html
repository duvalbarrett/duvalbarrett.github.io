Vintage Tech

Vintage Tech is a e-commerce website created to allow users to be able to shop for vintage phones and game consoles. It is a full-stack web application that allows login, purchasing and a delivery or pickup option.

Live Link

Project Overview

Vintage Tech is a e-commerce website created to allow users to be able to shop for vintage phones and game consoles such as the Blackberry, Nokia, and Sega Genesis. Vintage Tech is a great website for individuals to feel nostalgic and re-live their younger days by having a vintage phone or game console. If you ever feel like you’re getting too old, just remember you’re never too old, you’re just vintage. Vintage Tech is a full-stack web application that allows the user to create an account or sign in with their own google account securely using Auth0.  The users are able to add and remove items in their cart and can choose either a pickup or delivery option for purchase. This e-commerce website was made with React and JavaScript for the frontend and Nodejs, Express, Axios for the backend using a PostgreSQL database.
 
When a user clicks on a category they are able to see each phone and its price. When a user is finished viewing the items in that specific category they can return to the top of the page and in the header, they can navigate to the other category where they can see the items in that particular category along with their prices. The user can also click the Vintage Tech logo at the top of the page to be redirected to the homepage. When a user has found an item, they are interested they can sign in at the top right of the page either by creating an account or signing in securely using auth0. Once signed in the user is redirected to the cart where they can choose the number of items and on the right, it will show the user the total of their cart. The user can then choose either to checkout with either a delivery or pickup option. In the future I plan to implement the use of stripe for an easier checkout experience for users who choose the delivery option.
